# 股票投资监控系统需求文档

## 一、系统概述

### 1.1 投资框架
- 总投资额度：600万元人民币
- A股投资上限：50万元/只（个别可达100万元）
- 港股投资上限：150万元/只
- 核心策略：中轴价格仓位控制法
- 参考基准：财联社（主要）、东方财富、同花顺

### 1.2 核心策略说明
**中轴价格仓位控制法**
- 底仓：50%（长期持有，不轻易变动）
- 浮动仓：50%（根据价格波动动态调整）
- 触发阈值：股价较中轴上涨8%或下跌8%
- 调整幅度：每次调整浮动仓的20%

## 二、监控界面布局（四大区域）

### 2.1 自选股票显示区

#### 基础信息字段
| 字段 | 说明 |
|------|------|
| 股票代码 | 唯一标识 |
| 股票名称 | 显示名称 |
| 市场类型 | A股/港股 |
| 投资上限 | 最大投入金额 |
| 当前持仓数量 | 已购买股数 |
| 当前持仓成本 | 平均成本 |
| 当前市值 | 实时计算 |
| 浮动盈亏 | 实时盈亏 |

#### 策略核心字段
| 字段 | 说明 |
|------|------|
| 中轴价格 | 策略核心参考价 |
| 底仓金额 | 投资上限 x 50% |
| 浮动仓金额 | 投资上限 x 50% |
| 下次买入触发价 | 中轴 x 92% |
| 下次卖出触发价 | 中轴 x 108% |

#### 操作策略示例

**示例：阿里巴巴港股（09988.HK）**
- 投资上限：100万
- 中轴价格：100港元
- 底仓：50万（5000股）
- 浮动仓：50万（5000股）

**上涨场景（股价108港元，+8%）：**
- 触发：减持浮动仓20%
- 减持金额：50万 x 20% = 10万
- 实际卖出：约925股

**下跌场景（股价92港元，-8%）：**
- 触发：增持浮动仓20%
- 增持金额：50万 x 20% = 10万
- 实际买入：约1087股

### 2.2 自选股票市场情绪跟踪区

#### 监控内容
- 个股相关新闻实时推送
- 行业动态跟踪
- 机构研报评级变化
- 资金流向分析
- 技术指标变化（MACD、KDJ、均线等）

#### 信息来源（优先级）
1. 财联社（主要参考）
2. 东方财富
3. 同花顺
4. 雪球
5. 各大券商研报

#### 情绪评估输出
- 个股情绪指数（-100至+100）
- 每日市场情绪综合评估报告
- 个股策略建议（持有/增持/减持/观望）

---

### 2.3 市场热点与潜在预期跟踪区

#### 重点关注板块
| 板块类型 | 具体方向 |
|---------|---------|
| 科技板块 | 半导体、AI人工智能、新能源、机器人、5G/6G通信 |
| 金属类 | 贵金属（黄金、白银）、有色金属（铜、铝、锂、稀土）、小金属 |

#### 监控内容
- 板块资金流向
- 政策利好/利空消息
- 产业链上下游动态
- 国际市场联动
- 潜在预期事件（财报季、产品发布、政策发布）

---

### 2.4 综合策略建议区

#### 建议类型
- 买卖提醒（股价触发操作线）
- 中轴调整（基本面重大变化）
- 仓位调整（市场情绪极端）
- 板块轮动（热点切换）
- 风险预警（系统性风险）

## 三、实时数据要求

### 3.1 数据更新频率
| 数据类型 | 更新频率 |
|---------|---------|
| 股价数据 | 实时（秒级） |
| 持仓盈亏 | 实时计算 |
| 新闻推送 | 实时 |
| 市场情绪评估 | 每30分钟 |
| 热点板块扫描 | 每小时 |
| 日终报告 | 收盘后 |

### 3.2 重点提醒机制

#### 买卖提醒（最高优先级）
- 触发条件：股价达到中轴价格 +/- 8%
- 提醒方式：弹窗 + 声音 + 推送
- 提醒内容：股票名称、当前价格、建议操作、数量/金额

#### 新闻热点推送
- 重大利好/利空（立即推送）
- 行业政策变化（立即推送）
- 公司公告（立即推送）
- 机构评级调整（立即推送）

## 六、风险控制

### 6.1 单只股票风险
- 严格执行投资上限
- 底仓保护机制
- 止损线设置（-20%）

### 6.2 整体仓位风险
- 总仓位监控（不超过600万）
- 行业分散度检查
- 市场系统性风险预警

### 6.3 操作风险
- 买卖确认机制
- 交易时间限制
- 价格异常过滤

---

## 七、使用流程

### 7.1 初始化设置
1. 添加自选股，输入基本信息
2. 设置投资上限
3. 系统计算初始中轴价格
4. 确认底仓/浮动仓比例
5. 开启监控提醒

### 7.2 日常操作
1. 开盘前：查看隔夜新闻、国际市场动态
2. 交易中：接收实时提醒、查看操作建议
3. 触发买卖线：根据提醒执行交易
4. 收盘后：查看日终报告、调整策略参数

## 八、阿里巴巴港股策略示例

### 8.1 基本情况
- 标的：阿里巴巴-W（09988.HK）
- 最大投入资金：100万
- 已投入：70万
- 当前浮盈：20万

### 8.2 操作策略
- 中轴价格：由AI综合基本面、技术面、市场情绪动态计算
- 底仓比例：50%
- 浮动仓比例：50%
- 触发阈值：下跌5%买进10万，上涨8%卖出10万
- 参考因素：美股BABA走势、港股通资金流向

### 8.3 买卖触发示例
- 当前中轴价格假设为85港元
- 买入触发价：80.75港元（下跌5%）
- 卖出触发价：91.8港元（上涨8%）

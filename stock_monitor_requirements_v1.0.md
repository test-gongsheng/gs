# 股票投资监控系统 - 完善需求文档

## 一、系统概述

### 1.1 投资框架
| 项目 | 配置 |
|------|------|
| **总投资额度** | 600万元人民币 |
| **A股投资上限** | 50万元/只（个别可达100万元） |
| **港股投资上限** | 150万元/只 |
| **核心策略** | 中轴价格仓位控制法 + 价值投资 |
| **参考基准** | 财联社（主要）、东方财富、同花顺 |

### 1.2 策略融合说明
本系统融合两种策略：
1. **中轴价格仓位控制法**（通用策略）- 适用于所有持仓股票
2. **价值投资策略**（个股定制）- 针对特定股票可设定个性化参数

## 二、监控界面布局（四大区域）

### 2.1 自选股票显示区

#### 2.1.1 基础信息字段
| 字段 | 说明 | 示例 |
|------|------|------|
| 股票代码 | 唯一标识 | 09988.HK / 000001.SZ |
| 股票名称 | 显示名称 | 阿里巴巴-W |
| 市场类型 | A股/港股 | 港股 |
| 投资上限 | 最大投入金额 | 150万 |
| 当前持仓数量 | 已购买股数 | 5000股 |
| 当前持仓成本 | 平均成本 | 85.00元 |
| 当前市值 | 实时计算 | 90万 |
| 浮动盈亏 | 实时盈亏 | +5万 (+5.88%) |

#### 2.1.2 策略核心字段
| 字段 | 说明 | 计算方式 |
|------|------|---------|
| **中轴价格** | 策略核心参考价 | AI综合基本面+技术面+市场情绪动态计算 |
| **底仓金额** | 固定持仓部分 | 投资上限 × 底仓比例（默认50%） |
| **浮动仓金额** | 可调整部分 | 投资上限 × 浮动仓比例（默认50%） |
| **当前仓位比例** | 实际持仓/投资上限 | 实时计算 |
| **下次买入触发价** | 中轴 × (1 - 8%) | 如中轴100元 → 92元 |
| **下次卖出触发价** | 中轴 × (1 + 8%) | 如中轴100元 → 108元 |
| **距离触发点** | 当前价与触发价的差距 | 实时显示百分比 |

#### 2.1.3 操作策略详解

**【中轴价格仓位控制法】**

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
                    策略逻辑
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

基准：中轴价格（AI动态计算，定期评估调整）

仓位结构：
├─ 底仓：50%（长期持有，不轻易变动）
└─ 浮动仓：50%（根据价格波动动态调整）

浮动仓调整规则：
├─ 当股价较中轴上涨8% → 减持浮动仓的20%
├─ 当股价较中轴下跌8% → 增持浮动仓的20%
└─ 每次调整幅度：浮动仓金额的20%

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
                    示例演算
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

初始状态：
• 股票：阿里巴巴（09988.HK）
• 投资上限：150万元
• 中轴价格：100港元
• 底仓：75万（7500股）
• 浮动仓：75万（7500股）
• 当前持仓：15000股

【上涨场景】股价涨至108港元（+8%）：
→ 触发：减持浮动仓20%
→ 减持金额：75万 × 20% = 15万
→ 实际卖出：15万 ÷ 108 = 约1389股
→ 卖出后：底仓7500股 + 浮动仓(7500-1389)=6111股

【下跌场景】股价跌至92港元（-8%）：
→ 触发：增持浮动仓20%  
→ 增持金额：75万 × 20% = 15万
→ 实际买入：15万 ÷ 92 = 约1630股
→ 买入后：底仓7500股 + 浮动仓(7500+1630)=9130股

【连续触发】
• 上涨8%后再次上涨8%（相对中轴+16%）：
  → 继续减持浮动仓20%（基于剩余浮动仓）
• 下跌8%后再次下跌8%（相对中轴-16%）：
  → 继续增持浮动仓20%（基于剩余现金）
```

#### 2.1.4 个性化策略配置（进阶）

针对特定股票可设定个性化参数：

| 参数 | 默认值 | 可调整范围 | 说明 |
|------|--------|-----------|------|
| 底仓比例 | 50% | 30%-70% | 长期持有部分 |
| 浮动仓比例 | 50% | 30%-70% | 灵活调整部分 |
| 触发阈值 | 8% | 5%-15% | 买卖触发涨跌幅度 |
| 调整幅度 | 20% | 10%-50% | 每次调整浮动仓的比例 |
| 最大持仓比例 | 100% | 80%-120% | 相对投资上限的最大持仓 |
| 止损线 | -20% | -10%至-30% | 强制止损阈值 |

**【阿里巴巴港股示例】**
```
股票：09988.HK 阿里巴巴-W
投资上限：100万（原计划）/ 150万（港股上限）
已投入：70万
当前浮盈：20万

个性化策略：
• 底仓比例：60%（看好长期价值，底仓更高）
• 浮动仓比例：40%
• 触发阈值：下跌5%买入10万，上涨8%卖出10万
• 参考因素：美股BABA走势 + 港股通资金流向
```


## 八、风险控制体系

### 8.1 单只股票风险控制

| 风险类型 | 控制措施 | 触发条件 |
|---------|---------|---------|
| 过度集中 | 投资上限控制 | 单只A股≤50万，港股≤150万 |
| 深度套牢 | 止损线机制 | 股价跌破中轴-20%强制止损 |
| 追高杀跌 | 中轴锚定机制 | 只在±8%区间操作，不追涨杀跌 |
| 满仓操作 | 底仓保护机制 | 底仓50%长期持有，不轻易清仓 |

### 8.2 整体投资组合风险控制

| 风险类型 | 控制措施 | 触发条件 |
|---------|---------|---------|
| 总仓位过高 | 现金储备机制 | 总持仓不超过600万，保留应急资金 |
| 行业集中 | 行业分散度检查 | 单一行业不超过总仓位30% |
| 市场系统性风险 | 大盘预警机制 | 大盘暴跌>5%时提醒减仓 |
| 流动性风险 | 交易时间限制 | 非交易时间禁止大额操作 |

### 8.3 操作风险控制

| 风险类型 | 控制措施 |
|---------|---------|
| 误操作 | 买卖二次确认弹窗 |
| 数据错误 | 价格异常过滤（涨跌停价校验）|
| 网络延迟 | 本地缓存+离线提醒 |
| 情绪交易 | 策略信号强制执行建议 |

## 九、附录

### 9.1 术语表

| 术语 | 解释 |
|------|------|
| 中轴价格 | 策略核心参考价，基于基本面+技术面+情绪面综合计算 |
| 底仓 | 长期持有不轻易变动的仓位（默认50%） |
| 浮动仓 | 根据价格波动动态调整的仓位（默认50%） |
| 触发阈值 | 买卖操作的涨跌幅度阈值（默认8%） |
| 调整幅度 | 每次触发时调整浮动仓的比例（默认20%） |
| 投资上限 | 单只股票的最大投入金额 |

### 9.2 策略对比

| 策略 | 适用场景 | 特点 |
|------|---------|------|
| 中轴仓位控制法 | 震荡市、价值投资 | 高抛低吸，降低持仓成本 |
| 趋势跟踪法 | 牛市、成长股 | 追涨杀跌，获取趋势收益 |
| 定投法 | 长期持有、指数基金 | 分散时点风险，适合小白 |

# 股票监控技能 (Stock Monitor)

这个技能提供以下功能：

## 功能模块

### 1. 实时行情监控
- 获取A股主要指数实时行情（上证指数、深证成指、创业板指等）
- 监控个股实时价格
- 设置价格提醒

### 2. 技术分析
- 均线分析（MA5、MA10、MA20、MA60）
- MACD指标分析
- RSI指标分析

### 3. 市场情绪分析
- 整体市场情绪判断
- **板块情绪分析（东方财富行业分类）**
  - 金融类：银行、证券、保险、多元金融、金融科技
  - 房地产链：房地产开发、建筑装饰、建筑材料
  - 医药医疗：化学制药、生物制品、医疗器械、医疗服务、中药、医药商业
  - 大消费：白酒、饮料乳品、食品加工、家电、纺织服饰、商贸零售
  - 科技类：半导体、电子元件、消费电子、计算机设备、软件开发、通信设备、传媒
  - 新能源与电力：电力、光伏设备、风电设备、电池
  - 汽车与机械：汽车整车、汽车零部件、专用设备、通用设备
  - 军工航天：航天航空、船舶制造
  - 周期资源：贵金属、有色金属、煤炭行业、钢铁行业、石油行业、化学制品
  - 交通运输：物流、航运港口、航空机场
  - 公用环保：公用事业、环保
  - 农林牧渔：农牧饲渔
  - 轻工制造：造纸印刷
  - 综合类：综合
- **概念板块情绪（东方财富概念分类）**
  - 科技前沿：芯片概念、半导体概念、5G概念、人工智能、云计算、大数据、物联网、区块链
  - 新能源：新能源、锂电池、光伏概念、储能、氢能源、风电
  - 新能源汽车：新能源汽车、特斯拉、比亚迪概念、无人驾驶
  - 高端制造：机器人、工业母机、智能制造、航空航天
  - 医药医疗：创新药、CRO、医疗器械、医美
  - 数字经济：元宇宙、数字货币、云游戏、网红经济
  - 新基建：数据中心、东数西算、特高压、充电桩
  - 新消费：新零售、跨境电商、免税概念、预制菜、宠物经济、养老概念
  - 资源：稀土永磁、黄金概念、锂矿
  - 环保碳中和：碳中和、碳交易、节能环保
  - 国企改革：国企改革、央企改革、乡村振兴
  - 信息安全：网络安全、信创、国产替代
  - 市场风格：科创板、北交所、次新股

### 4. 财报分析
- 获取上市公司财务数据
- 关键财务指标分析（营收、净利润、ROE等）
- 财务健康评分

## 使用方法

### 查看大盘行情
```bash
python3 monitor.py index
```

### 查询个股
```bash
python3 monitor.py stock -c 000001
```

### 技术分析
```bash
python3 monitor.py tech -c 000001
```

### 市场情绪（包含板块分析）
```bash
python3 monitor.py sentiment
```

### 单独查看板块情绪
```bash
python3 monitor.py sector
```

### 财报分析
```bash
python3 monitor.py finance -c 000001
```

## 情绪指标说明
- 🔥 强势 (涨幅 ≥ 3%)
- 📈 活跃 (涨幅 1.5% ~ 3%)
- 🙂 平稳 (涨幅 0% ~ 1.5%)
- 😐 偏弱 (跌幅 -1.5% ~ 0%)
- 📉 低迷 (跌幅 -3% ~ -1.5%)
- ❄️ 弱势 (跌幅 < -3%)

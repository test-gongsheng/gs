# 2026-03-16 记忆

## 股票监控系统中轴价格问题修复

### 问题发现
用户发现润泽科技（300442）的中轴价格显示为 90.51 元，但正确的应该是约 63.19 元。
- 2025年12月17日收盘价 53 元
- 当前价格约 82 元
- 中轴价格计算结果应为 63.19 元

### 问题根因
1. **导入时并发请求超时**：`confirmImport()` 使用 `Promise.all` 并发请求所有股票的中轴价格 API，导致部分请求超时失败
2. **失败回退机制**：API 失败时回退到使用成本价作为中轴价格
3. **浏览器缓存**：JS 文件被浏览器缓存，导致代码更新不生效

### 修复方案
1. **顺序执行 API 请求**：将 `Promise.all` 改为 `for` 循环顺序执行，避免并发超时
2. **版本号机制**：添加 `APP_VERSION` 常量，版本不匹配时自动清除 localStorage
3. **强制刷新缓存**：通过 URL 参数 `?v=10` 强制浏览器加载最新 JS

### 代码改动
- `static/js/import.js`：修改 `confirmImport()` 函数
- `static/js/app.js`：添加版本检查和 `refreshAxisPrices()` 函数
- `templates/index.html`：更新 JS 引用版本号

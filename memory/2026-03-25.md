## 北向资金数据源失效修复

**问题**：北向资金显示为 0 亿

**根因**：AKShare 的 `stock_hsgt_hist_em` 接口返回 NaN（数据源失效）

**修复**：
- 弃用 AKShare 接口
- 后端改用 Session 复用 HTTP 连接
- 北向资金暂时显示"待接入"（需接入 Wind/同花顺 iFinD 等付费 API）
- 前端适配：资金为 0 且状态为"待接入"时显示提示而非 "+0亿"

**提交**：`656cc93`

## 港股个股沽空数据不显示修复

**问题**：选中港股时看不到个股沽空数据

**根因**：`renderStockDetail` 函数中港股判断只显示简化提示，**没有调用** `renderHKShortRiskWarning()` 来渲染完整的个股沽空数据

**修复**：将简化代码替换为调用 `renderHKShortRiskWarning()`，现在会显示：
- 个股昨日沽空数据（沽空股数、沽空比率）
- 沽空趋势（1周/1月/3月变化）
- 市场整体沽空数据

**提交**：`ed9d16f`

## 港股沽空数据闪烁修复

**问题**：选中港股后，沽空股数在"加载中"和具体数据之间闪烁

**根因**：`renderStockDetail()` 每次价格更新都会调用，每次都重置"加载中"状态

**修复**：
- 添加5分钟缓存机制 (`_hkShortCache`)
- 分离渲染逻辑为 `renderHKShortDataInternal()` 函数
- 有缓存时直接使用缓存数据，不再显示"加载中"

**提交**：`80a6b26`

## 港股个股沽空趋势数据

**问题**：个股沽空趋势变化（1周/1月/3月）没有显示

**根因**：`get_hk_stock_short_selling` 函数没有返回 `changes` 字段

**修复**：
- 后端计算历史变化趋势（对比1周前、1月前、3月前的数据）
- 返回 `changes` 字段，包含 `volume_change` 和 `ratio_change`
- 前端已支持显示这些数据

**示例数据**（腾讯00700）：
- 1周变化：+425.81万股 (+12.1%)
- 3月变化：+469.05万股 (+8.29%)

**提交**：`5cf3b88`

# 南向资金数据设计方案

## 一、需求分析

### 1. 实时南向资金数据
- **整体南向资金净流入**：港股通（沪+深）当日净流入金额
- **更新频率**：交易时段每3分钟刷新

### 2. 历史数据（90个有效交易日）
- **南向资金整体趋势**：过去90个交易日的每日净流入
- **持仓个股南向数据**：腾讯、阿里、比亚迪电子的每日南向持股变化

### 3. 数据展示
- **线性图表**：使用 ECharts 展示资金流向趋势
- **关键指标**：累计净流入、30日均值、趋势方向

---

## 二、数据源设计

### 1. 南向资金整体数据
**数据源**：akshare `stock_hsgt_hist_em(symbol="南向资金")`
- 字段：`日期`, `当日成交净买额`, `买入成交额`, `卖出成交额`, `历史累计净买额`
- 频率：日级别，T+1更新（港交所披露机制）

**实时数据补充**：akshare `stock_hsgt_fund_flow_summary_em()`
- 字段：`成交净买额`, `资金净流入`
- 频率：当日实时（交易时段）

### 2. 港股通个股数据
**数据源**：akshare `stock_hsgt_individual_em(symbol="港股代码")`
- 字段：`持股日期`, `持股数量`, `持股市值`, `当日收盘价`
- 注意：这是沪深股通数据（外资买A股），不是港股通数据

**替代方案**：东方财富数据中心 API
- 直接爬取东方财富港股通持股页面
- URL: `https://data.eastmoney.com/hkstock/ggt.html`

## 四、前端设计

### 1. 南向资金概览卡片
位置：市场情绪面板新增卡片

内容：
- 实时净流入（大字体，红绿颜色）
- 较昨日变化（百分比）
- 30日/60日/90日均值
- 趋势指示箭头

### 2. 南向资金趋势线图
位置：市场情绪面板新增图表

配置：
- X轴：日期（最近90个交易日）
- Y轴：净流入金额（亿港元）
- 系列1：每日净流入（柱状图，红绿）
- 系列2：累计净流入（折线，蓝色）
- 系列3：30日均线（虚线）

交互：
- 悬停显示当日详细数据
- 支持缩放查看局部区间
- 点击日期跳转该日市场回顾

### 3. 港股个股南向资金图表
位置：股票详情面板新增标签页

内容：
- 90日南向持股数量变化曲线
- 持股占港股总股本比例变化
- 关键指标：5日变化、30日变化、趋势判断

## 五、数据更新策略

### 1. 服务端缓存
```python
# 南向资金整体数据缓存
_southbound_cache = {
    'realtime': {...},
    'history': [...],
    'timestamp': 0
}
CACHE_TTL_REALTIME = 180  # 实时数据3分钟缓存
CACHE_TTL_HISTORY = 3600  # 历史数据1小时缓存
```

### 2. 数据获取时机
- **交易时段（9:30-16:00）**：每3分钟刷新实时数据
- **收盘后**：获取当日最终数据，更新历史记录
- **日初**：预加载前90日历史数据

### 3. 数据存储
- **Redis/Memory**：实时数据缓存
- **本地JSON文件**：历史数据持久化（每日更新）
- **数据库（可选）**：长期历史归档

## 七、数据源注意事项

### 1. 港交所披露机制
- T+1日披露上一交易日数据
- 每日收盘后约19:00更新
- 实时数据需要付费API（Wind/同花顺）

### 2. 数据质量
- 节假日无数据（自动跳过）
- 半日市数据可能存在异常值
- 需要数据清洗和异常值处理

### 3. 备选方案
- 主数据源：东方财富/akshare
- 备用数据源：新浪财经、腾讯财经
- 降级方案：仅展示昨日数据+提示
